## Gig Description

**Stop wasting hours fighting AWS configs. I will deploy your Node.js application on AWS with production-grade infrastructure, so you can focus on building your product.**

I am Saad Sohail, a Senior Full-Stack Engineer with 6+ years of experience deploying and scaling Node.js applications on AWS. I have built and maintained cloud infrastructure for production SaaS platforms serving thousands of users, and I bring that same battle-tested approach to every deployment I deliver.

### What You Get

Every deployment follows a repeatable, documented process:

1. **Server provisioning** — EC2 instance sized to your traffic, configured with security groups and key pairs.
2. **Reverse proxy** — Nginx configured for your domain with gzip compression and proper headers.
3. **SSL termination** — Free Let's Encrypt certificate with auto-renewal, enforcing HTTPS across your entire application.
4. **Process management** — PM2 cluster mode for zero-downtime restarts and automatic crash recovery.
5. **Database connectivity** — Secure connection to RDS (PostgreSQL or MySQL) or your existing database.
6. **CI/CD automation** — Push to GitHub and your code deploys automatically through AWS CodePipeline and CodeDeploy.

## FAQ

**Q: Do I need an existing AWS account?**
A: Yes. You will need an active AWS account with billing enabled. If you do not have one, I can walk you through creating it during our kickoff call. AWS costs (EC2, RDS, etc.) are billed directly to your account and are separate from my service fee.

**Q: What Node.js frameworks do you support?**
A: I deploy Express, NestJS, Fastify, Koa, and any standard Node.js application. If your project uses a less common framework, message me first and I will confirm compatibility.

**Q: Will you handle domain and DNS setup?**
A: Yes. I will configure Route 53 or update your existing DNS provider to point your domain to the new server with proper A/CNAME records.

**Q: What if my app needs environment variables or secrets?**
A: I configure all environment variables securely on the server using PM2 ecosystem files or AWS Systems Manager Parameter Store for the Premium tier. Your secrets never touch version control.

**Q: Can you deploy Docker containers instead of bare Node.js?**
A: Absolutely. If your project is already containerized, I will deploy using Docker Compose on EC2 or configure ECS Fargate for the Premium tier. Message me to discuss the best approach.

**Q: Do you offer ongoing maintenance after deployment?**
A: This gig covers initial deployment and setup. For ongoing monitoring, patching, and infrastructure management, I offer a separate monthly maintenance package. Ask me for details.
